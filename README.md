# Android Studio
<b>Позволяет создавать нативные приложения для Android OS<br>Android - открытая операционная система, то есть свободно модефицируемая</b>

## Устройство платформы Android
<b>С точки зрения архитектуры Android можно выделить следующие уровни:
1) Базовый уровень(linux Kernel) - уровень абстракции между аппаратным уровнем и программами пользователя
2) Набор библиотек и среда использования(Libraries and Android Runtime) обеспечивает важнейшую базовую функциональность для приложения, содержит виртуальную машину Dalvik и базовые библиотеки Java, необходимые для запуска Android приложений
3) Уровень каркаса приложений(Application Framework) обеспечивает разработчикам доступ к API, предоставляемым компонентами системы уровнея библиотек
4) Уровень приложения (Application) - набор предустановленных базовых приложений.</b>

### Все Android-приложения имеют расширения .apk(Android package)

<b>JDK</b> - бесплатно распространяемый комплект разработчика приложений на языке Java, включающий в себя компилятор Java, стандартные библиотеки классов Java, примеры, документацию, различные утилит ыи исполнительную систему Java Runtime Environment. В состав JDK не входит интегрированная среда разработки(IDE)


<b>Android Sofware Development Kit(SDK)</b> - содежрит множество инструментов и утилит для создания и тестирования приложений

## API
<b>Интерфейс, который обеспечивает связь между двумя разными "частями" программного обеспечения</b>
